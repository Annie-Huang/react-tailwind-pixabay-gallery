C:\react>create-react-app react-tailwind-pixabay-gallery
C:\react\react-tailwind-pixabay-gallery>npm install tailwindcss postcss-cli autoprefixer -D
C:\react\react-tailwind-pixabay-gallery>npx tailwind init tailwind.js --full

The tailwind.css file will be used by us to import Tailwind CSS styles, and for custom configurations and rules.
The main.css will hold the styles that are generated as a result of what we have in the tailwind.css file.

C:\react\react-tailwind-pixabay-gallery>npm start
C:\react\react-tailwind-pixabay-gallery>npm run build

Whenever you add something into .env file, you will have to restart the server.

You will need to check the C:\react\ChromeDevTool.txt on how to do the slow network
